Some examples using the curl tool

span>a>Getting all the inventory data

h3> 
                curl -X GET 'localhost:8080/items'code>
pre>

which returns something like:

p> 
                {"Inventory": [{"amount": 15, "date": "2014-10-05", "category": "Fruit", "name":
                    "Banana", "id": 1, "location": "Amsterdam"}]}code>
pre>

pre>

returns:

                {"item": [{"amount": 15, "date": "2014-10-05", "category": "Fruit", "name": "Banana",
                    "id": 1, "location": "Amsterdam"}]}code>

                    "id": 3, "location": "Peru"}]}code>
