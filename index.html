On the first line is the document type. Then inside the <head> tag there are <meta> tags specifying content type, generator, creation date, and changed date information. Then there is a <style> tag, empty title. The <body> tag specifies language and direction (left-to-right in this case). HTTP stands for HyperText Transfer Protocol. It is used for communication between web client and web server. It uses port 80. Source: Computer Networks - Andrew Tannenbaum
SSH

FTP
FTP stands for File Transfer Protocol. It is used to transferring files between networked computers. It uses port 21. Source: Computer Networks - Andrew Tannenbaum

Question 6

<strong> is preferrable in this case. <b> is meant to draw attention without changing the mood of the sentence and strong is meant to convery importance or urgency (source:

https://html.spec.whatwg.org/multipage/semantics.html). For example, compare
<strong> Do not</strong> eat that; it's poisonous.
to
<b> This</b>, on the other hand, is not poisonous.

Another pair of similar HTML constructs are <em> and <i>. <em> is meant to emphasize the highlighted text and <i> is meant to change the mood of it, loosely speaking.

Question 7

Divs are block elements and spans are inline elements. This means that on a webpage there will be a line break after a div and not after a span. They are similar in that they both denote sections of content (source: http://www.1keydata.com/css-tutorial/div-span.php).
